# Resumo Estruturado: 
☁️ **O que é computação em nuvem?**
Computação em nuvem é a entrega de serviços de tecnollogia através da Internet, uso de servidores remotos para armazenar, gerenciar e processar dados ao invés de depender de um servidor local.

⚙️ Modelos de Serviço:
- **Iaas - Infraestutura como serviço**
> Oferece infraestrutura virtualizada (como máquina virtual e armazenamento).
- **Paas - Plataforma como serviço**
> Fornece ambiente para desenvolvimento e deploy de aplicações.
- **Saas - Software como serviço**
> Disponibiliza softwares prontos.

⚙️ Modelos de Implatação: 
- **Nuvem Pública**: Recursos compartilhados pela internet.
- **Nuvem Privada**: Infraestutura exclusiva de uma organização ou grupo.
- **Nuvem Híbrida**: Combinação entre nuvem pública e privada.

# 📚 Glossário: 
| **Termo** | **Definição** |
|---|---|
| Cloud Computing | Computação em nuvem |
| Iaas |  Infraestutura como serviço |
| Paas  |  Plataforma como serviço |
| Saas  | Software como serviço |
| VM | Máquina Virtual |
| Virtualização | Criação de ambientes virtuais |
